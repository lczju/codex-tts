第一阶段默认优先选择：

```text
标贝中文标准女声音库（DataBaker / Biaobei）
```

原因：

- 中文普通话。
- 单说话人女声。
- 约 12 小时。
- 专业录音棚录制。
- 48kHz / 16-bit / mono / PCM WAV。
- 有文本标注和韵律相关标注。
- 比多说话人大数据集更适合作为 VITS 第一版 fine-tuning 数据。

注意：

- 该数据集官方说明为非商用。
- 如果后续项目要商用，必须重新确认授权，不能默认可商用。

## 3. 下载顺序建议

### 第一优先级：标贝中文标准女声音库

用途：

- VITS 中文单说话人 fine-tuning。
- 中文网页展示样例。
- 后续和 CosyVoice 结果做主观对比。

推荐处理方式：

```text
先下载完整数据集，然后只抽取 1 到 2 小时用于第一轮实验。
```

原因：

- 全量 12 小时不算太大，`D:` 盘可以承受。
- 但第一轮没必要全部训练。
- 先抽样能降低云端上传时间和 GPU 成本。

### 第二优先级：AISHELL-3

用途：

- 后续多说话人 TTS。
- 说话人相似度、零样本或少样本实验。

不建议第一轮直接使用全量 AISHELL-3，原因是：

- 下载包约 19GB。
- 解压后体积更大。
- 多说话人处理流程比单说话人复杂。

### 第三优先级：LJSpeech

用途：

- 英文 TTS baseline。
- 验证 VITS 英文训练流程。

它不适合作为当前中文 TTS 主线，但它的优势是：

- 下载包约 2.6GB。
- 单说话人。
- 文件结构清晰。
- 公开使用限制少。


结论：

- 下载 LJSpeech 2.6GB：空间足够。
- 下载 AISHELL-3 19GB：空间也足够，但暂不建议第一阶段使用。

建议预留空间：

```text

AISHELL-3：建议预留 60GB

```

预留空间包含：

- 原始压缩包。
- 解压后的 wav。
- 处理后的 wav。
- metadata。
- 抽样子集。
- 临时文件。

## 5. 本项目推荐落盘位置

下载压缩包：

```text
datasets/downloads/
```

解压后的原始数据：

```text
datasets/raw/public/
```

抽样后的第一轮训练数据：

```text
datasets/raw/speaker_a/
```

处理后的训练数据：

```text
datasets/processed/speaker_a/
```

## 6. 第一轮实验数据量

即使下载了完整标贝数据集，第一轮也建议只抽取：

```text
1 到 2 小时
```

优先挑选：

- 音频长度 3 到 12 秒。
- 文本干净。
- 没有明显噪声。
- 没有过长停顿。
- 没有异常标点或特殊符号。
